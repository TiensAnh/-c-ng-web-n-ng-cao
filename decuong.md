# ĐỀ CƯƠNG CHỨC NĂNG DỰ KIẾN CỦA ĐỀ TÀI

## Tên đề tài

Xây dựng Website đặt tour du lịch trực tuyến

## 1. Mô tả đề tài

Website đặt tour du lịch trực tuyến giúp người dùng tìm kiếm, lựa chọn
và đặt tour một cách nhanh chóng, thuận tiện.\
Hệ thống hỗ trợ quản lý tour, khách hàng, đơn đặt tour và thanh toán,
giúp nâng cao hiệu quả quản lý và cải thiện trải nghiệm người dùng.

## 2. Mục tiêu đề tài

-   Xây dựng hệ thống website đặt tour du lịch trực tuyến hoàn chỉnh\
-   Hỗ trợ người dùng tìm kiếm và đặt tour dễ dàng\
-   Giúp quản trị viên quản lý tour và đơn đặt hiệu quả\
-   Tối ưu trải nghiệm người dùng với giao diện thân thiện\
-   Tích hợp chức năng nâng cao

## 3. Các chức năng dự kiến

### 3.1 Admin / Quản trị viên

-   Đăng nhập / Đăng xuất\
-   Quản lý người dùng\
-   Quản lý tour\
-   Quản lý danh mục\
-   Quản lý đơn đặt tour\
-   Quản lý thanh toán\
-   Thống kê báo cáo

### 3.2 User / Khách hàng

-   Đăng ký / Đăng nhập\
-   Xem & tìm kiếm tour\
-   Xem chi tiết tour\
-   Đặt tour\
-   Thanh toán\
-   Xem lịch sử\
-   Hủy tour theo điều kiện

## 4. Chức năng nâng cao

-   Chatbot tư vấn tour\
-   Đánh giá tour\
-   Yêu thích tour\
-   Gửi email xác nhận

## 5. Công nghệ sử dụng

-   Backend: Node.js + Express\
-   Frontend: HTML, CSS, JavaScript\
-   Database: MySQL

## 6. Kết quả mong đợi

-   Website hoạt động đầy đủ\
-   Giao diện thân thiện\
-   Hệ thống ổn định, dễ mở rộng
